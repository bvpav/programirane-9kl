# Добре дошли в наръчника за допринасяне <3

## Стъпки за предлагане на промени

1. [Създайте fork на GitHub хранилището](https://github.com/bvpav/programirane-9kl/fork)
2. Запазете промените във вашия fork.
3. [Отворете pull request в GitHub](https://github.com/bvpav/programirane-9kl/compare)

Ако ви се налага да правите редакции твърде често, свържете се с мен, или [отворете issue](https://github.com/bvpav/programirane-9kl/issues/new/choose) и ще ви дам директен commit достъп.

## Редактиране на документите

Цялото съдържание се състои в `src/` подпапката. Текстът е записан в [Markdown формат](https://rust-lang.github.io/mdBook/format/markdown.html) и е компилиран в сайт с [mdBook](https://rust-lang.github.io/mdBook).

Всеки документ трябва да бъде включен в [src/SUMMARY.md](./src/SUMMARY.md)

### Стил на форматиране

Моля при редакциите си, спазвайте следните конвенции:

- Всеки файл с материал от презентация започва със заглавието ѝ, последвано от линк към оригиналната презентация.
- Използвайте заглавие от 1 ниво за името на презентацията (`#`)
- Използвайте заглавие от 2 ниво за заглавието на слайдовете (`##`)
- Използвайте заглавие от 3 ниво за въпросите от слайдовете (`###`)
- Заградете оператори и идентификатори в обратни кавички (Пример: `` `return`, `+`, `/==` ``)
- Заградете цитиран C код в отварящи ` ```c ` и затварящи ` ``` `, _на отделни редове_.

### Признание за помощ

Моля след редакциите си, добавете вашето име и/или псевдоним в [src/avtori.md](./src/avtori.md). Може и по желание да включите линк към всякакво място в интернет, което притежавате, като социална мрежа или уебсайт.

### Локален преглед на редакциите

Сайтът използва [mdBook](https://rust-lang.github.io/mdBook/). Можете да го прегледате локално, следвайки следните стъпки:

1. Инсталирайте mdBook:

Използвайте [инструкциите в документацията на mdBook](https://rust-lang.github.io/mdBook/guide/installation.html) за да го инсталирате.

2. Клонирайте GitHub хранилището:

```shell
$ git clone https://github.com/bvpav/programirane-9kl.git
$ cd programirane-9kl
```

> Моля копирайте командите без `$`.

3. Отворете сайта:

```shell
$ mdbook serve --open
```

Тази команда _би трябвало_ да отвори системния уеб браузър на адрес <http://localhost:3000>, където можете да прегледате локалната версия на документите.

При проблеми и за повече информация относно работата с mdBook, моля допитайте се до [официалната ѝ документация](https://rust-lang.github.io/mdBook/).
